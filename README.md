# Stack Overflow Developer Survey Dashboard - IBM Capstone Project
## Survey Analytics Project
This repository contains files related to the Developer Survey Analysis dashboard created as part of the IBM Data Analytics Capstone Project.

## Contents
- `dashboard.pdf` / `screenshots/` : Exported images of the Cognos dashboard visualizations
- `data/` : Optional sample data used for analysis (if allowed to share)
- `scripts/` : Python scripts used for data cleaning, analysis, and visualization
- `README.md` : This file

## Overview
The project analyzes survey data to uncover trends in:
- Programming languages and frameworks
- Database usage
- Work experience, job satisfaction, and compensation

Visualizations include bar charts, stacked charts, bubble plots, and histograms to highlight key insights.

## How to Use
- View exported dashboard images in the `screenshots/` folder or PDF.
- Optionally, run the Python scripts in `scripts/` to reproduce the analysis (requires Python, pandas, matplotlib, etc.).
