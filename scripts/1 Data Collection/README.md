Four notebooks included in this section:

### **1. `CollectingDataUsingApi.ipynb`**

Covers how to collect data from an online API. Topics include:

* Making API requests
* Parsing JSON responses
* Converting API results into structured data formats (DataFrames)
* Saving API data for later use

---

### **2. `ReviewWebScraping.ipynb`**

A review notebook for key web scraping concepts:

* Basic HTML structure
* Using BeautifulSoup to parse HTML
* Extracting specific elements (tables, text, attributes)

---

### **3. `WebScraping.ipynb`**

Exercise in web scraping. Demonstrates how to:

* Request webpage content
* Parse web pages using BeautifulSoup
* Extract structured data from tables and lists
* Clean and store scraped data

---

### **4. `ExploringDataSet.ipynb`**

Introduces initial data exploration techniques, including:

* Loading CSV or scraped datasets
* Summary statistics
* Identifying patterns, anomalies, and missing values
* Basic visualizations for quick insights
